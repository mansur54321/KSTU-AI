# AdGuard Privacy (Chrome Extension)

<img width="1024" height="559" alt="image" src="https://github.com/user-attachments/assets/6a22d5dc-df47-4a41-84c9-213664bc94f8" />

Расширение для Chrome, которое помогает проходить онлайн-тесты, оставаясь полностью незаметным. Использует **Google Gemini 3 flash** для анализа вопросов и изображений, автоматически обходит системы защиты и предоставляет объяснения ответов.

## Основные возможности

### Искусственный интеллект
*   **Multi-Model System:** Gemini 3 flash с авто-переключением при ошибках 503/429.
*   **Vision API:** Распознает изображения в вопросах и вариантах ответов.
*   **System Prompt:** Отвечает на том же языке, что и вопрос.

### Стелс-режим
*   Расширение выглядит как обычный блокировщик рекламы (AdGuard).
*   На странице теста — минимум видимых следов: только полупрозрачный индикатор и серые точки-маркеры.
*   API-ключи скрыты от DevTools страницы (запросы идут через background).

### Обход защиты (Anti-Cheat)
*   Снимает запрет на **выделение текста**.
*   Включает **правую кнопку мыши**.
*   Разрешает **копирование и вставку**.

### Авто-ключ из Univer
*   Впишите API-ключ в поле Email на Univer в формате: `AIzaSy...вашключ@aa.aa`
*   Расширение автоматически найдёт и сохранит ключ при загрузке страницы.
*   Обычные email-адреса игнорируются — ключ извлекается только если строка до `@` совпадает с форматом Gemini API-ключа.

### Управление
*   **Alt + S:** Решение всего теста.
*   **Alt + Клик:** Решение одного вопроса.

## Установка

1. Скачайте [последний релиз](https://github.com/mansur54321/KSTU-AI/releases).
2. Откройте Chrome: `chrome://extensions/`
3. Включите **Режим разработчика** (справа сверху).
4. Нажмите **"Загрузить распакованное"**.
5. Выберите папку с расширением.

---

## Настройка

**Способ 1 — через Univer (рекомендуется):**
1. Получите бесплатный API-ключ в [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Зайдите в профиль на Univer (`student/mydata/edit/`).
3. Впишите ключ в поле Email: `AIzaSy...вашключ@aa.aa`
4. Нажмите "Сохранить". Ключ подхватится автоматически.

**Способ 2 — через настройки расширения:**
1. Нажмите на иконку расширения → шестерёнка.
2. Вставьте ключ в поле "API КЛЮЧИ".
3. Нажмите **"Сохранить"** → **"Проверить"**.

---

## Как использовать

1. Откройте страницу с тестом.
2. Блокировки снимаются автоматически при загрузке.
3. Нажмите **`Alt + S`** — ИИ решит весь тест.
4. Рядом с правильными ответами появятся серые точки `•` — наведите курсор, чтобы увидеть объяснение.
5. Если включён автоклик — ответ проставится сам.
